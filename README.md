# LSPD:MP Website

## Description
Website for LSPD:MP server on GTA5's multiplayer platform alt:V

## Setup

Run :
```bash
npm i
```

Create file `.env.local` at root and populate the file with those filled vars: 
```env
VITE_DISCORD_INVITE=<discord invite of server>
```

## Build

Run :
```bash
npm run build
```
